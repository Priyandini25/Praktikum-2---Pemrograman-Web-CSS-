Pengalaman Orgamisasi

Pengalaman Organisasi saya selama Kuliah ini adalah saya pernah menjadi Kepala Divisi Konsumsi dan Perlengkapan pada salah satu Program Kerja Himasifo yaitu NOF2019 (Night Of Friendship)

Saat ini kegiatan aktif organisasi saya adalah sebagai Delegasi 2019 di BLJ Himasifo (Badan Legislatif Jurusan). juga saya sedang menjabat sebagai Sekretaris BLJ Himasifo untuk TA 2020/2021.

Dan ketika masa jabatan saya berakhir tahun ini, saya mencalonkan kembali sebagai Delegasi 2019 BLJ Himasifo untuk TA 2021/2022. Alhamdulillah saya terpilih kembali, berkat kepercayaan dari Tuhan YME dan juga teman-teman angkatan saya.
